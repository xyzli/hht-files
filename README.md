Application of DSDLI algorithm on OCT scans of telangiectasia. 
https://doi.org/10.1002/jbio.201600264

Labels variance of speckle pattern as moving blood to delineate vasculature within the lesion.
Semantic segmentation and feature extaction, including: vascular fraction, epidermal fraction, fractal dimension and vasculature over lesional value (VALVe).
Voxels can be generated in ImageJ with slice data.

Post-processing with median and Gaussian filters.
